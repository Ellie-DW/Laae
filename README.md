# 🍁 메이플 가게부

메이플스토리 메소 수입·지출을 기록하고 관리하는 가계부 웹앱입니다.

## 기능

- **수입/지출 기록** — 보스, 거래, 장비, 주문서, 스타포스, 큐브 등 카테고리별 기록
- **메소 단위 입력** — `1.5B`, `500M`, `30K` 형식 지원
- **대시보드** — 잔액, 총 수입/지출, 거래 건수 한눈에 확인
- **거래 내역** — 날짜별 그룹, 검색, 필터 기능
- **카테고리 통계** — 수입/지출 비율 시각화
- **캐릭터 관리** — 여러 캐릭터별 기록 가능
- **로컬 저장** — 브라우저 localStorage에 자동 저장

## 시작하기

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:5173` 접속

## 빌드

```bash
npm run build
```
